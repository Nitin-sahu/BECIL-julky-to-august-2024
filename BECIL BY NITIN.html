What is HTML?

HTML stands for Hypertext Markup Language. It is used to define the structure of web pages, including headings, text, paragraphs, multimedia, and other elements.

Types of Tags

There are two types of tags:

  1. Start tag (or open tag)
  2. End tag (or close tag)
